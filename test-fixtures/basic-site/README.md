# Basic Test Site Fixture

This is a fixture site used for integration testing of jekyll-ts.

## Structure

- `index.md` - Homepage with default layout
- `about.md` - About page with custom permalink
- `_posts/` - Blog posts with categories and tags
- `_layouts/` - Page layouts (default, post with inheritance)
- `_includes/` - Reusable components (header, footer)
- `_recipes/` - Custom collection for testing
- `assets/` - Static files (CSS)

## Features Tested

- Markdown rendering
- Liquid templating
- Layout application and inheritance
- Includes
- Collections
- Posts with categories and tags
- Custom permalinks
- Static file copying
- Site and page variables
